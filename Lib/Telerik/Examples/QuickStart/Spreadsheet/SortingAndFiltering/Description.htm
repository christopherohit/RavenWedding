Filtering in RadSpreadsheet allows you to focus on a specific set of your data and sorting lets you arrange your data in a convenient way. You can filter your records using value, dynamic, top 10 and custom filters and sort the data by value, fill color, font color in ascending or descending order, or even using a custom list. Feel free to arrange the order log using the predefined views below or set up your own custom filters and sort conditions.
