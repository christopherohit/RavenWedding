CellFormatting and RowFormatting events are fired when the state of the respective element is changed. You can use these events to change the style of elements based on certain conditions. You can find additional information on this topic in our online documentation: CellFormatting and RowFormatting

*Due to the specifics of the RadVirtualGrid the CellFormatting event is fired for all cells of a given row and only after that is the RowFormatting for this row fired.

**The data presented in the example is randomly generated.
