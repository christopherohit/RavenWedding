This example demonstrates the drill down functionality of RadPivotGrid for LocalDataSourceProvider.

The extraction of the records from the LocalDataSourceProvider ItemsSource that are used for accumulating the result for a specific row and column groups is done via calling the LocalDataSourceProvider.GetUnderlyingData method. The IEnumerable result is passed in the event arguments of the LocalDataSourceProvider.GetUnderlyingDataCompleted event.

Double click on a RadPivotGrid value cell to see its drill down data displayed in a RadGridView.
