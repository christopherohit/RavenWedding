The Telerik RadMenu creates attractive and flexible menus, designed to be easy for users to use, but with the flexibility to include advanced features such as multi-line text, top-level icons, icons above text, and more. The current example demostrates the multiline text capability of RadMenu. It is achieved through the Text property, where you enter multiline item text. Alternatively, you can set the multiline text programmatically.
