The Telerik RadRibbonBar provides an easy-to-design implementation of the Microsoft Office 2007 ribbon user interface. With a RadRibbonBar you can organize all of the functionality of your application into a single compact ribbon. It features easily customizable visual appearance through the intuitive Visual Style Builder; Consistent design interface; Support for unlimited number of nested controls; Backstage View Microsoft Office 2007 user interface; Gallery Items; and Contextual Tabs Key Tips (keyboard navigation specific for the RibbonBar control).
