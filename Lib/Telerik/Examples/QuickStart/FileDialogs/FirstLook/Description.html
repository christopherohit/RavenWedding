Updated: The new features of File Dialogs are: Brand new BreadCrumb control, which supports text editing and auto-complete, support for environment variables, Custom Places in the navigation tree view to allow adding predefined places. File dialogs are UI controls, designed to enable users to open a file, select a folder and save a file. This example demonstrate how you can use all available types of dialogs, such as RadOpenFileDialog, RadOpenFolderDialog and RadSaveFileDialog.
