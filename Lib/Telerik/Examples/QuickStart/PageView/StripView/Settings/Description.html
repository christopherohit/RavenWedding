The example allows different strip view settings to be modified: Strip alignment defines the position of the strip within the view. Item alignment defines the position of items rectangle within the strip. Item fit mode determines whether the items will fit within the available strip's client rectangle. Item size mode defines rules to be applied when calculating each item's size. Item content orientation specifies how items will be oriented, depending on the current strip's alignment. Item spacing defines the space between each two items within the strip.
