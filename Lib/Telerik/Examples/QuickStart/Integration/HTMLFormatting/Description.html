Telerik® UI for WinForms provide an advanced text styling mechanism, which can be applied to all Telerik products, because it enhances the smallest element in the Telerik Presentation Framework - the text primitive. The new rich text formatting mechanism uses plain HTML tags to display formatted text such as font style, font color, fontSystem.Drawing.Size options and simple layouts.
To turn on Html-like formatting the text must start with an <html> tag; use <size=[+|-]value> to set fontSystem.Drawing.Size use, and <br> to create new line feed. To bold, underline and italic text, use the corresponding opening and closing tags. Font family is set through <font=Family>.
