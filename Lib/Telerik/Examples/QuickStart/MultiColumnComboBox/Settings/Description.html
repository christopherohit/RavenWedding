RadMultiColumnComboBox is a modified RadDropDownList hosting a RadGridView in its drop-down. This example shows the RadMultiColumnComboBox control and its ability to filter its rows. When the AutoFilter property is set to true and an appropriate filtering expression is assigned to the RadGridView. Filtering is applied as you type and different auto complete modes are supported - Suggest, Append and SuggestAppend. The example also shows right-to-left language support.
