The Object-relational hierarchy mode of RadGridView allows the grid to be bound to data/business objects and entities, as well as object relational mapping frameworks, such as Telerik’s OpenAccess ORM and Entity Framework.

The approach is extremely simple – once you have created your model, simply set is as the GridView’s DataSource and set the AutoGenerateHierarchy property to true; RadGridView will automatically take care of the rest.
