This example demonstrates the functionality of the filtering row. The available filter functions are Equals, Starts with, Ends with, Contains, Less than, Greater than, etc. The logical operation applied between the individual filters are AND - a row will appear in the result set if it satisfies all cell filter conditions; OR - a row will appear in the result set if it satisfies at least one cell filter condition, and None - no filtering will be applied.
