Use the CellValidating and RowValidating events to validate data in RadGridView. These events fire when a cell(row) loses input focus, enabling content validation. Canceling the validation will not allow the user to leave the current cell or row.

Set the ErrorText property of the row to indicate validation errors. This will turn on the error indicator inside the row header column. Hover it to see the message.
