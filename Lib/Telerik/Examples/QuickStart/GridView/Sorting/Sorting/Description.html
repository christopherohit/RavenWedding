Three sorting directions are supported by RadGridView: Ascending, Descending, and None. As in Excel, sorting is allied by clicking on the column header text. This example demonstrates how data can be sorted programmatically by using the GridSortExpression class that provides the sorting options for RadGridView by using the SortExpressions collection of the GridTableView class, and how to set the two different sorting modes - single or multi-column sorting.

You can find additional information on this topic in our online documentation
