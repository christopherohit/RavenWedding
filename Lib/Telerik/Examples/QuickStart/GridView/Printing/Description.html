You can use the built-in printing functionality of RadGridView to print its data. This example demostrates how to directly print a grid, show the RadPrintPreviewDialog or show the Print settings dialog. The printing supportof RadGridView is capable of printing: grouped grid, summaries, hidden rows, hidden columns, headers on each page, view definitions. You can set page headers, page footers, watermarks and paper size settings.

You can find additional information on this topic in our online documentation
