This example demonstrates the following events: CurrentRowChanging (fires whenever a change to the current record is attempted and allows cancellation (in this example it cancels selection of row with id = 3)), CurrentRowChanged (fires whenever the current record is changed), CommandCellClick (fires on a click in each command cell), CellFormatting (fires for each cell to allow formatting of the cell), OnClick, OnDoubleClick, MouseDown, MouseUp, and CellMouseMove events.
