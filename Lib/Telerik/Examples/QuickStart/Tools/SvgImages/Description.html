This example demonstrates the difference in scaling when using raster images and vector images. Scalable Vector Graphics (SVG) defines vector-based graphics in XML format. SVG file can be scaled to different sizes without losing quality so they scale appropriately on all DPI settings. You can apply SVG image on a control just by using the SvgImage property that is integrated in LightVisualElement and ImagePrimitive.
