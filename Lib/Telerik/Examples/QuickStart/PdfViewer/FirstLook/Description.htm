Updated RadPdfViewer is now optimized to work together with RadPdfProcessing library model. RadPDFViewer arms you with the ability to visualize PDF documents right within your WinForms application without the need of any external viewer. It comes with a predefined intuitive UI that provides you with every mean to utilize the features of this control in a quick and easy manner. The default viewer’s UI buttons allow you to zoom-in or zoom-out, pan and scroll the document or select a fragment of text. The control utilizes virtualization in order to guarantee stable performance with large documents.
