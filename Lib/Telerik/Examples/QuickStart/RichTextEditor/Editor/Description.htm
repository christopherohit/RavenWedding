RadRichTextEditor offers broad editing and formatting capabilities. This example demonstrates how RadRichTextBox can provide MS Work-like editing experience. Support for import/export in .docx, .html, .pdf (export only), .rtf, .xaml, .txt formats is also available.

In the R3 2019 release, RadRichTextEditor has been updated with new Paste Options.
