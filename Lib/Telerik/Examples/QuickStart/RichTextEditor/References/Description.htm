Using References you can create a more structured and easy to navigate document. In this way, you will no only enrich its content, but also improve its readability. Footnotes and endnotes are particularity useful when remarks have to be added to the document, but inserting them inline would interrupt the natural flow of thought. The Citations and Bibliography functionality is also used in a similar manner in scenarios such as when creating a thesis, a dissertation or another document that requires sources of information to be quoted. The feature is enhanced by the presence of predefined styles, which are designed especially for this purpose: APA, ISO690 and Harvard-Angila.
