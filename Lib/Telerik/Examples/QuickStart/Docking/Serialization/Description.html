This example demonstrates RadDock's serialization and deserialization functionality. Several ToolWindows are already created and docked for the purpose of the example. Use Save from the menu to serialize the RadDock structure: dockingManager1.SaveXML(FileName);
or use Load to load the RadDock structure: dockingManager1.LoadXML(FileName);
Also, you can reset the dockManager by calling: dockingManager1.Reset();

You can find additional information on this topic in our online documentation
