This example shows a gallery of vector images displayed with RadPictureBox controls with limited end user functionality - pan, zoom and context menu are disabled. When you click on an image, a fully functional RadPictureBox is opened - pan, zoom, editing, context menu are now enabled. RadPictureBox is used to display images in client applications with no dependencies on external APIs. It supports various image processing tools and effects like zoom, pan, and a set of predefined panels where you can add your buttons or other custom elements.
